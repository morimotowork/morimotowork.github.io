Art Collage (Beta/2019)

Website gerador de ideias para criação de colagens. O usuário indica quais materiais têm disponível para trabalhar e o nível de criatividade em que se encontra. Com base nisso, o site cria uma lista combinando critérios aleatórios para a composição.

Também foi desenvolvida uma identidade visual dinâmica para o projeto. Trabalho parte da minha graduação-sanduíche na Faculdade de Belas Artes (U. Porto).


Acesse a versão beta (disponível em português e inglês)

Fotografias da sessão de validação com usuários que foi conduzida para avaliar a usabilidade do website. Cada participante gerou uma lista de ideias com a plataforma a partir do seu próprio dispositivo (telefone, tablet ou laptop), e então foi instruído a produzir uma colagem a partir de tais critérios. Revistas, jornais e papéis coloridos estavam entre os materiais fornecidos para a criação.
